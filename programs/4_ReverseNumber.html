Write a program to reverse a given number.

he reverse of a number is obtained by reversing the order of its digits. For example, if the input number is 12345, the reversed number would be 54321. This involves repeatedly extracting the last digit of the number and building a new number by shifting previously extracted digits to the left.

Number 12345: Reverse of 12345 is 54321.

Number 1000: Reverse of 1000 is 1 (leading zeros are omitted).

Java Program


import java.util.Scanner;

public class ReverseNumber {

    public static void main(String[] args) {
        // Create a Scanner object to read input from the user
        Scanner scanner = new Scanner(System.in);

        // Prompt the user to enter a number
        System.out.print("Enter a number: ");
        int number = scanner.nextInt();

        // Initialize a variable to store the reverse of the number
        int reverse = 0;

        // Make a copy of the input number to preserve the original value
        int temp = number;

        // Loop to reverse the number
        while (temp != 0) {
            int lastDigit = temp % 10;  // Get the last digit of the number
            reverse = reverse * 10 + lastDigit;  // Build the reversed number
            temp /= 10;  // Remove the last digit from the number
        }

        // Print the reversed number
        System.out.println("The reverse of " + number + " is " + reverse + ".");

        // Close the scanner to free up resources
        scanner.close();
    }
}

Solution Description

This Java program reads an integer from the user and calculates its reverse by extracting its digits one at a time from the end using modulus and division operations. It constructs the reversed number by multiplying the current reverse by 10 and adding the last digit in each iteration, effectively shifting the previous digits to the left. The loop continues until all digits are processed. This method efficiently handles positive and negative numbers and outputs the reversed number, preserving the original sign.
