Write a program to check whether the given number is a spy Number.

A spy number is a number where the sum of its digits is equal to the product of its digits.

Sum of digits = Product of digits
Consider the number 1124.
The sum of its digits is 1+1+2+4=8
The product of its digits is 1×1×2×4=8.
Since the sum and the product are equal, 1124 is a spy number.

Java Program


import java.util.Scanner;

public class SpyNumberChecker {
    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        // Prompt the user to enter a number
        System.out.print("Enter a number: ");
        int number = scanner.nextInt();

        // Initialize variables to store the sum and product of digits
        int sumOfDigits = 0;
        int productOfDigits = 1;
        int temp = number; // Temporary variable to hold the original number

        // Loop through each digit of the number
        while (temp > 0) {
            int digit = temp % 10; // Get the last digit of the number
            sumOfDigits += digit;  // Add the digit to the sum
            productOfDigits *= digit; // Multiply the digit to the product
            temp /= 10; // Remove the last digit from the number
        }

        // Check if the number is a spy number
        if (sumOfDigits == productOfDigits) {
            System.out.println(number + " is a spy number.");
        } else {
            System.out.println(number + " is not a spy number.");
        }

        // Close the scanner
        scanner.close();
    }
}

Solution Description

This Java program checks if a number is a spy number by calculating the sum and product of its digits. The program prompts the user to enter an integer, iterates through each digit to compute the sum and product, and then compares the results. If the sum and product are equal, the number is identified as a spy number; otherwise, it is not. The program efficiently uses a loop to extract digits by repeatedly taking the remainder and integer division by ten.
