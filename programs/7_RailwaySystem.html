Design a class RailwayTicket with following description.

Instance variables/data members.

String name : To store the name of the customer.

String coach : To store the type of coach customer wants to travel.

long mobno : To store customer’s mobile number.

int amt : To store basic amount of ticket.

int totalamt : To store the amount to be paid after updating the original amount.

Member methods :

void accept () – To take input for name, coach, mobile number and amount.

void update() – To update the amount as per the coach selected.

extra amount to be added in the amount as follows)

Type of Coaches Amount

First_AC = 700

Second_AC = 500

Third_AC = 250

sleeper = None

void display() – To display all details of a customer such as name, coach, total amount and mobile number.

Write a main method to create an object of the class and call the above member methods.


import java.util.Scanner;

class RailwayTicket {
    // Instance variables
    private String name;
    private String coach;
    private long mobno;
    private int amt;
    private int totalamt;

    // Method to accept details from the user
    public void accept() {
        Scanner sc = new Scanner(System.in);

        // Input name, coach, mobile number, and amount
        System.out.print("Enter name: ");
        name = sc.nextLine();
        System.out.print("Enter coach (First_AC/Second_AC/Third_AC/sleeper): ");
        coach = sc.nextLine();
        System.out.print("Enter mobile number: ");
        mobno = sc.nextLong();
        System.out.print("Enter basic amount of ticket: ");
        amt = sc.nextInt();

        // Initialize total amount to the base amount initially
        totalamt = amt;
    }

    // Method to update the total amount based on the type of coach
    public void update() {
        // Add extra charges based on the selected coach
        switch (coach) {
            case "First_AC":
                totalamt += 700; // Add Rs. 700 for First_AC
                break;
            case "Second_AC":
                totalamt += 500; // Add Rs. 500 for Second_AC
                break;
            case "Third_AC":
                totalamt += 250; // Add Rs. 250 for Third_AC
                break;
            case "sleeper":
                // No extra charge for sleeper class
                break;
            default:
                System.out.println("Invalid coach type selected!");
                break;
        }
    }

    // Method to display the ticket details
    public void display() {
        System.out.println("Customer Details:");
        System.out.println("Name: " + name);
        System.out.println("Coach: " + coach);
        System.out.println("Mobile Number: " + mobno);
        System.out.println("Total Amount to be Paid: Rs. " + totalamt);
    }

    // Main method to test the class
    public static void main(String[] args) {
        // Create object of RailwayTicket class
        RailwayTicket ticket = new RailwayTicket();

        // Call the member methods
        ticket.accept();  // Take input from the user
        ticket.update();  // Update the amount based on the coach type
        ticket.display(); // Display the ticket details
    }
}
