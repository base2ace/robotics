Write a program to check if a string is a Palindrome.

A palindrome is a word, phrase, number, or sequence of characters that reads the same forward and backward, ignoring spaces, punctuation, and capitalization. In the context of this problem, we are focusing on strings. A string is considered a palindrome if it is identical when read from left to right and from right to left.

"racecar" is a palindrome because it reads the same backward as forward.

"madam" is a palindrome because it reads the same backward as forward.

Java Program


import java.util.Scanner;

public class PalindromeChecker {

    public static void main(String[] args) {
        // Create a Scanner object to read input from the user
        Scanner scanner = new Scanner(System.in);

        // Prompt the user to enter a string
        System.out.print("Enter a string: ");
        String inputString = scanner.nextLine();

        // Convert the input string to lowercase to make the palindrome check case-insensitive
        String normalizedString = inputString.toLowerCase();

        // Initialize variables to use as pointers
        int left = 0;
        int right = normalizedString.length() - 1;

        // Flag to track whether the string is a palindrome
        boolean isPalindrome = true;

        // Loop to check characters from both ends of the string
        while (left < right) {
            // Check if the characters at left and right pointers are the same
            if (normalizedString.charAt(left) != normalizedString.charAt(right)) {
                isPalindrome = false; // Characters don't match, not a palindrome
                break; // Exit the loop as we found a mismatch
            }
            // Move the pointers towards the center
            left++;
            right--;
        }

        // Print the result based on the palindrome check
        if (isPalindrome) {
            System.out.println(inputString + " is a palindrome.");
        } else {
            System.out.println(inputString + " is not a palindrome.");
        }

        // Close the scanner to free up resources
        scanner.close();
    }
}

Solution Description

This Java program checks if a given string is a palindrome by comparing characters from the beginning and end of the string moving towards the center. The program reads a string input from the user, converts it to lowercase to ensure case insensitivity, and then uses two pointers to iterate from both ends of the string towards the middle. It checks if the characters at these positions match, flagging the string as not a palindrome if any mismatch is found. If the string passes all character comparisons without mismatches, it is identified as a palindrome, and the result is displayed.
