Write a program that allows a user to input 20 integer elements into an array. Once the array is populated, the program should perform the following operations.

1. Display the largest number from the array.

2. Display the smallest number from the array.

3. Display the sum of all the elements of the array.

Java Program


import java.util.Scanner;

public class ArrayOperations {

    public static void main(String[] args) {
        // Create a Scanner object to read input from the user
        Scanner scanner = new Scanner(System.in);

        // Define the size of the array
        final int SIZE = 20;

        // Create an array to store the integer elements
        int[] numbers = new int[SIZE];

        // Prompt the user to input 20 integers
        System.out.println("Enter " + SIZE + " integer elements:");

        // Read elements into the array
        for (int i = 0; i < SIZE; i++) {
            numbers[i] = scanner.nextInt();
        }

        // Initialize variables for largest, smallest, and sum
        int largest = numbers[0];
        int smallest = numbers[0];
        int sum = 0;

        // Iterate over the array to find the largest, smallest, and calculate the sum
        for (int i = 0; i < SIZE; i++) {
            if (numbers[i] > largest) {
                largest = numbers[i]; // Update largest if current element is greater
            }
            if (numbers[i] < smallest) {
                smallest = numbers[i]; // Update smallest if current element is smaller
            }
            sum += numbers[i]; // Add current element to the sum
        }

        // Display the results
        System.out.println("Largest number: " + largest);
        System.out.println("Smallest number: " + smallest);
        System.out.println("Sum of all elements: " + sum);

        // Close the scanner to free up resources
        scanner.close();
    }
}

Solution Description

This Java program reads 20 integer elements from the user into an array and performs three operations: finding the largest number, finding the smallest number, and calculating the sum of all elements. The program initializes the largest and smallest values using the first element of the array and iteratively updates these values as it processes each element. The sum of all elements is computed by adding each element to a cumulative sum variable. The results are displayed after all elements are processed. This approach efficiently calculates the desired results with a single pass through the array.
